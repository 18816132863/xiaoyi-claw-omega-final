------------------------------------------------------------
  4. 生产环境部署步骤
------------------------------------------------------------

方式一: Docker Compose

```bash
# 1. 启动 PostgreSQL 和 Redis
docker-compose up -d postgres redis

# 2. 等待服务就绪
docker-compose ps
docker-compose logs postgres
docker-compose logs redis

# 3. 运行数据库迁移
docker-compose exec postgres psql -U openclaw -f /migrations/001_task_system.sql

# 4. 启动应用服务
docker-compose up -d message-server task-daemon

# 5. 验证服务
curl http://localhost:18790/health
docker-compose logs task-daemon
```

方式二: 云服务

```bash
# 1. 设置环境变量
export DATABASE_URL="postgresql://user:pass@host:5432/db"
export REDIS_URL="redis://host:6379/0"

# 2. 运行迁移
python3 scripts/run_migrations.py

# 3. 启动服务
python3 scripts/message_server.py --port 18790 &
python3 scripts/task_daemon.py &

# 4. 验证
curl http://localhost:18790/health
```
