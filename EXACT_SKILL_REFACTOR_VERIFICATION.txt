  三、真实输出样例

### schedule_task 创建真实任务

```python
result = await schedule_task.execute({
    "task_type": "once",
    "message": "Test task",
    "user_id": "test"
})

# 返回:
{
    "success": True,
    "task_id": "8206c8f3-580c-42f2-aef3-0695c2535aba",
    "task_type": "once",
    "status": "persisted",
    "created_at": "2026-04-23T23:20:00"
}
```

### export_history 返回样例

```python
result = await export_history.execute({"task_id": "xxx"})

# 返回:
{
    "success": True,
    "task_id": "xxx",
    "task": {...},  # 真实任务记录
    "runs": [...],  # 真实运行记录
    "steps": [...], # 真实步骤记录
    "events": [...], # 真实事件记录

}
```

### replay_run 返回样例

```python
result = await replay_run.execute({"run_id": "yyy"})

# 返回:
{
    "success": True,
    "run_id": "yyy",

```
