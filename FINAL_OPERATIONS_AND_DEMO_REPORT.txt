### 1.2 NOTIFICATION 授权闭环

**文件**: `scripts/check_notification_auth.py`

**输出三态**:
- `configured` - authCode 已配置且有效
- `not_configured` - authCode 未配置
- `invalid` - authCode 已配置但无效

**配置位置**:
1. 环境变量 `XIAOYI_AUTH_CODE`
2. `config/xiaoyi_config.json`
3. `.xiaoyi_auth`
4. `~/.xiaoyi/auth_code`

### 1.3 失败率/超时率分析

**新增命令**: `invocation_audit_cli.py breakdown`

**输出**:
- 按 Capability 统计
- 按 Error Code 统计 (Top 10)
- 按 Capability + Error Code 统计

### 1.4 一键演示

**新增文件**:
- `scripts/demo_bootstrap.py` - 演示环境初始化
- `scripts/demo_run_all.sh` - 一键运行所有演示

**流程**:
1. 初始化数据库
2. 预热演示数据
3. 检查授权状态
4. 显示统计
5. 运行健康巡检
6. 导出报告

### 1.5 CLI 增强
