## 二、核心改动

### 2.1 L4 风险策略修正

**原口径**: L4 默认禁止或强确认

**新口径**: L4 默认进入**强确认模式**，不是默认禁止

| 等级 | 类型 | 默认策略 |
|---|---|---|
| L0 | 查询、总结、解释 | 自动执行 |
| L1 | 轻写入 | 自动执行 |
| L2 | 短信、通知、批量 | 策略控制 + 限流 |
| L3 | 删除、拨电话、重要修改 | 默认确认 |
| L4 | 支付、金融、账号、隐私、游戏自动化 | **强确认 + 分步执行 + 全程审计** |
| BLOCKED | 违法、盗号、绕过反作弊、规避风控 | 拒绝 |

### 2.2 BLOCKED 场景定义

只有以下场景才进入 BLOCKED：
- illegal_activity - 违法活动
- bypass_anti_cheat - 绕过反作弊
- account_theft - 盗号
- unauthorized_payment - 未授权支付
- batch_harassment - 批量骚扰
- bypass_risk_control - 规避风控
- competitive_cheating - 竞技作弊
- resource_farming_bot - 脚本挂机刷资源
- unauthorized_trading - 未授权交易

### 2.3 游戏策略修正

**允许**:
- 视觉辅助
- 策略建议
- 练习模式
- 单机非竞争
- 复盘分析

**强确认 (L4)**:
- 自动点击
- 长时间操作
- 资源/道具/抽卡操作

**禁止 (BLOCKED)**:
- 联网竞技自动代打
- 绕过反作弊
- 挂机刷资源
- 未授权交易

### 2.4 学习闭环

**第一次执行**:
- 记录目标、计划、能力、技能
- 记录成功/失败步骤
- 记录耗时、确认次数
- 记录视觉路径

**第二次相似任务**:
- 复用成功路径
- 避开失败路径
- 按成功率排序技能
- 减少不必要追问
- **L4 仍需强确认，不因学习而越权**
