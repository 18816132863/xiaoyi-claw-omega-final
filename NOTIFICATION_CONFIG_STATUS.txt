### 解决方案

需要用户重新获取授权码：

1. 从手机桌面右滑进入负一屏
2. 点击左上角头像
3. 进入"我的"页面，点击右上角设置图标
4. 选择"动态管理"
5. 点击"关联账号"
6. 找到"Claw 智能体"并点击获取授权码

### 配置命令

获取新授权码后，使用以下命令配置：

```bash
openclaw config set skills.entries.today-task.config.authCode YOUR_NEW_AUTH_CODE
```
