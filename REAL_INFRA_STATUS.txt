------------------------------------------------------------
  解决方案
------------------------------------------------------------

方案一: 在有权限的环境中运行

需要以下条件之一:
- 有 sudo 权限的 Linux 服务器
- 有 Docker 的环境
- 有云上 PostgreSQL/Redis 实例

部署步骤:
```bash
# 1. 安装服务
sudo apt-get install postgresql redis-server

# 2. 启动服务
sudo systemctl start postgresql redis

# 3. 创建数据库
sudo -u postgres createdb openclaw

# 4. 运行迁移
psql -U postgres -d openclaw -f infrastructure/storage/migrations/001_task_system.sql

# 5. 配置环境变量
export DATABASE_URL="postgresql://postgres@localhost/openclaw"
export REDIS_URL="redis://localhost:6379/0"

# 6. 运行测试
python3 scripts/real_infra_test.py
```

方案二: 使用 Docker Compose

```bash
# 1. 启动服务
docker-compose up -d postgres redis

# 2. 等待就绪
docker-compose ps

# 3. 运行测试
docker-compose exec task-daemon python3 scripts/real_infra_test.py
```

方案三: 连接云服务

```bash
# 1. 配置云服务连接
export DATABASE_URL="postgresql://user:pass@host:5432/db"
export REDIS_URL="redis://host:6379/0"

# 2. 运行测试
python3 scripts/real_infra_test.py
```
