# AGENTS.md V2.9.3 - 六层架构 + 文件保护深度整合

---

## 每次会话

1. `SOUL.md` — 身份
2. `USER.md` — 用户
3. `TOOLS.md` — 工具规则
4. `memory/YYYY-MM-DD.md` — 今日日记

---

## 六层架构 (固定不变)

```
L1: 核心认知层 → 身份、规则、引导、配置
L2: 记忆上下文层 → 记忆存储、检索、日记
L3: 任务编排层 → 路由、工作流、调度、状态机
L4: 能力执行层 → 技能执行、结果验证、健康检查
L5: 稳定治理层 → 安全、权限、审计、文件保护
L6: 基础设施层 → 路径、插件、性能、注册表
```

---

## 层间调用协调

```
请求流: 用户请求 → L1解析 → L3路由 → L4执行 → L5验证 → 返回结果
保护流: 删除请求 → L5文件保护 → 人工确认 → L6执行 → L5审计日志
记忆流: L1触发词 → L2加载记忆 → L3编排 → L4执行 → L2存储结果
```

### 调用规则

| 层级 | 可调用 |
|------|--------|
| L1 | L2, L3, L4, L5, L6 (触发) |
| L2 | L6 (查找技能) |
| L3 | L2, L4, L5, L6 |
| L4 | L2, L5 |
| L5 | L2, L6 |
| L6 | L2, L5 |

## L5 稳定治理层 (深度整合)

### 模块列表

| 模块 | 作用 | 入站 | 出站 |
|------|------|------|------|
| security.py | 安全检查 | L1,L3,L4,L6 | audit.py |
| permissions.py | 权限管理 | L1,L3,L4,L6 | audit.py |
| audit.py | 审计日志 | 全部层 | memory |
| **file_guardian.py** | 文件保护 | L1,L3,L4,L6 | audit.py, paths.py |
| **protected_files.json** | 保护列表 | file_guardian | - |

### 文件保护流程

```
┌─────────────────────────────────────────────────────────────┐
│  Step 1: 任何层发起删除请求                                  │
│          → 调用 L5/file_guardian.request_delete()           │
├─────────────────────────────────────────────────────────────┤
│  Step 2: file_guardian 检查 protected_files.json            │
│          → 判断是否受保护                                    │
├─────────────────────────────────────────────────────────────┤
│  Step 3: 显示文件描述和必要性                                │
│          → 说明文件作用、层级、大小                          │
├─────────────────────────────────────────────────────────────┤
│  Step 4: 等待人工确认                                        │
│          → 用户输入 "确认删除 [文件名]"                      │
├─────────────────────────────────────────────────────────────┤
│  Step 5: 确认后调用 L6/paths.py 解析路径                     │
│          → 验证路径安全性                                    │
├─────────────────────────────────────────────────────────────┤
│  Step 6: 移至 .trash 目录 (不用 rm)                          │
│          → 保留文件备份                                      │
├─────────────────────────────────────────────────────────────┤
│  Step 7: 调用 L5/audit.py 记录日志                           │
│          → 永久记录删除操作                                  │
└─────────────────────────────────────────────────────────────┘
```

## 安全规则

- **禁止**禁用 execution-validator
- **禁止**绕过 file_guardian 删除文件
- **禁止**直接使用 rm 删除文件
- **使用** trash 替代 rm
- **安装技能前**检查 skill-scope
- **敏感操作**需确认
- **删除文件**需二次确认 + 说明必要性
