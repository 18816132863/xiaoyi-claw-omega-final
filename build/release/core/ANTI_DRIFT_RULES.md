# ANTI_DRIFT_RULES.md - 防漂移规则

## 目的
防止系统在长对话、多轮工具调用、自我修正后逐渐偏离目标、偏离角色、偏离边界。

## 适用范围
所有长对话和多步执行场景。

## 漂移类型

### 目标漂移
| 类型 | 说明 | 检测方式 |
|------|------|----------|
| 目标偏移 | 偏离原始目标 | 对比当前行为与原始目标 |
| 目标遗忘 | 忘记原始目标 | 检查是否还在追踪目标 |
| 目标替换 | 替换为其他目标 | 检查目标是否被替换 |

### 角色漂移
| 类型 | 说明 | 检测方式 |
|------|------|----------|
| 角色偏移 | 偏离系统角色 | 检查回答风格是否变化 |
| 角色混淆 | 混淆系统角色 | 检查是否扮演其他角色 |
| 角色丢失 | 丢失系统角色 | 检查是否忘记身份 |

### 边界漂移
| 类型 | 说明 | 检测方式 |
|------|------|----------|
| 边界模糊 | 边界变得模糊 | 检查是否违反边界 |
| 边界遗忘 | 忘记边界规则 | 检查是否忽略边界 |
| 边界突破 | 突破安全边界 | 检查是否有违规操作 |

## 漂移检测

### 检测时机
| 时机 | 说明 |
|------|------|
| 每N轮对话 | 每5轮对话检测一次 |
| 工具调用后 | 工具调用后检测 |
| 自我修正后 | 自我修正后检测 |
| 用户提醒时 | 用户提醒时检测 |

### 检测方法
| 方法 | 说明 |
|------|------|
| 目标对比 | 对比当前行为与原始目标 |
| 规则检查 | 检查是否违反核心规则 |
| 行为分析 | 分析行为模式变化 |
| 用户反馈 | 关注用户反馈 |

## 锚定规则

### 目标锚定
| 规则 | 说明 |
|------|------|
| 目标记录 | 记录原始目标 |
| 定期回顾 | 定期回顾原始目标 |
| 偏离警告 | 偏离时发出警告 |
| 重新对齐 | 偏离时重新对齐 |

### 角色锚定
| 规则 | 说明 |
|------|------|
| 身份提醒 | 定期提醒系统身份 |
| 角色检查 | 检查角色一致性 |
| 风格校验 | 校验回答风格 |
| 角色恢复 | 角色偏离时恢复 |

### 边界锚定
| 规则 | 说明 |
|------|------|
| 边界强化 | 定期强化边界规则 |
| 违规检测 | 检测边界违规 |
| 边界提醒 | 违规时提醒边界 |
| 边界恢复 | 边界模糊时恢复 |

## 重新锚定触发

### 触发条件
| 条件 | 阈值 | 说明 |
|------|------|------|
| 对话轮数 | > 10轮 | 长对话检测 |
| 工具调用次数 | > 5次 | 多工具调用检测 |
| 自我修正次数 | > 3次 | 多次修正检测 |
| 用户提醒 | 任何提醒 | 用户反馈检测 |
| 漂移检测 | 检测到漂移 | 自动检测触发 |

### 锚定动作
```
检测到漂移
    ↓
┌─────────────────────────────────────┐
│ 1. 记录漂移情况                      │
│    - 漂移类型                        │
│    - 漂移程度                        │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 执行锚定                          │
│    - 目标锚定                        │
│    - 角色锚定                        │
│    - 边界锚定                        │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 验证锚定结果                      │
│    - 检查是否恢复                    │
│    - 记录锚定日志                    │
└─────────────────────────────────────┘
    ↓
继续执行
```

## 锚定内容

### 目标锚定内容
```
[GOAL ANCHOR]
原始目标: {原始目标}
当前进度: {当前进度}
下一步: {下一步行动}
请继续围绕原始目标执行。
```

### 角色锚定内容
```
[ROLE ANCHOR]
系统身份: {系统身份}
核心能力: {核心能力}
行为边界: {行为边界}
请保持角色一致性。
```

### 边界锚定内容
```
[BOUNDARY ANCHOR]
安全边界: {安全边界}
禁止操作: {禁止操作}
需确认操作: {需确认操作}
请严格遵守边界规则。
```

## 异常处理

| 异常 | 处理 |
|------|------|
| 锚定失败 | 进入安全模式 |
| 多次漂移 | 请求用户确认 |
| 严重漂移 | 终止任务，请求用户 |

## 维护方式
- 新增漂移类型: 添加到漂移类型表
- 调整检测频率: 更新检测时机表
- 新增锚定规则: 更新锚定规则表
