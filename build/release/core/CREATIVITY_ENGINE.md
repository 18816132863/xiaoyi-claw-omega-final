# CREATIVITY_ENGINE.md - 创意生成引擎

## 目的
生成原创创意，激发创新思维。

## 核心能力

### 1. 原创创意生成
- 产品创意
- 营销创意
- 设计创意
- 内容创意
- 商业创意

### 2. 创意组合
- 融合组合
- 类比迁移
- 变异生成
- 重组创新

### 3. 创意评估
- 新颖性评估
- 可行性评估
- 价值评估
- 影响力评估

### 4. 创意优化
- 迭代优化
- 反馈学习
- 持续改进

## 使用示例

```bash
# 生成产品创意
openclaw creative "为健身APP生成新功能创意"

# 生成营销创意
openclaw creative "为新品发布生成营销创意"

# 创意评估
openclaw creative evaluate "创意描述"

# 创意优化
openclaw creative optimize "创意描述"
```
