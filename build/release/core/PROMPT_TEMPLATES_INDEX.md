## 模板格式

### 标准格式
```
[TEMPLATE: {模板ID}]
名称: {模板名称}
用途: {模板用途}
变量: {变量列表}
---
{模板内容}
---
[END TEMPLATE]
```

### 变量说明
| 变量格式 | 说明 |
|----------|------|
| {{var}} | 必需变量 |
| {{var?default}} | 可选变量，有默认值 |
| {{var*}} | 列表变量 |

## 模板示例

### 搜索模板示例
```
[TEMPLATE: search_001]
名称: 基础搜索
用途: 执行基础搜索任务
变量: query, max_results
---
请搜索以下内容：{{query}}

要求：
- 返回最多 {{max_results?10}} 条结果
- 按相关性排序
- 标注来源
---
[END TEMPLATE]
```

### 拒绝模板示例
```
[TEMPLATE: refuse_001]
名称: 安全拒绝
用途: 安全原因拒绝请求
变量: reason, alternative
---
抱歉，我无法执行这个请求。

原因：{{reason}}

{{alternative?如果您有其他问题，我很乐意帮助您。}}
---
[END TEMPLATE]
```

### 澄清模板示例
```
[TEMPLATE: clarify_001]
名称: 意图澄清
用途: 澄清用户意图
变量: options
---
我需要确认一下您的需求，请问您是想要：

{{options}}

请告诉我您的选择。
---
[END TEMPLATE]
```

## 模板管理

### 版本控制
| 操作 | 说明 |
|------|------|
| 新增模板 | 分配新ID |
| 修改模板 | 更新版本号 |
| 废弃模板 | 标记废弃 |

### 质量检查
| 检查项 | 说明 |
|--------|------|
| 变量完整 | 所有变量都有说明 |
| 格式正确 | 格式符合规范 |
| 内容清晰 | 内容表达清晰 |

## 异常处理

| 异常 | 处理 |
|------|------|
| 模板不存在 | 使用默认模板 |
| 变量缺失 | 使用默认值 |
| 格式错误 | 自动修正 |

## 维护方式
- 新增模板: 按格式添加
- 修改模板: 更新版本
- 废弃模板: 标记废弃
