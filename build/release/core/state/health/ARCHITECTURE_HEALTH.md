# ARCHITECTURE_HEALTH.md - 架构健康度评估

## 目的
评估和监控架构健康度。

## 评估指标

| 指标 | 目标 | 权重 |
|------|------|------|
| 耦合度 | < 0.3 | 20% |
| 内聚度 | > 0.7 | 20% |
| 复杂度 | < 20 | 15% |
| 测试覆盖率 | > 80% | 15% |
| 技术债务 | < 100h | 15% |
| 可维护性 | > 80 | 15% |

## 核心能力

### 1. 健康度评分
- 综合评分
- 分项评分
- 趋势分析

### 2. 债务分析
- 债务识别
- 债务量化
- 优先级排序

### 3. 改进建议
- 自动建议
- 优先级排序
- 实施路径

## 使用示例

```bash
# 评估架构健康度
openclaw arch health

# 分析技术债务
openclaw arch debt

# 获取改进建议
openclaw arch suggest

# 生成报告
openclaw arch report --format html
```
