# TUTORIAL.md - 教程系统

## 目的
提供场景化教程，帮助用户掌握各项能力。

## 教程分类

| 教程 | 内容 | 时长 |
|------|------|------|
| 基础教程 | 启动、交互、任务 | 15分钟 |
| 个人模式 | 对话、知识、规划 | 20分钟 |
| 企业模式 | 协作、项目、架构 | 30分钟 |
| 高级教程 | 多智能体、代码、自动化 | 45分钟 |

## 核心特性

### 1. 交互式
- 边学边练
- 实时反馈

### 2. 动手实践
- 真实场景练习
- 即时验证

### 3. 测验验证
- 知识点测验
- 技能验证

### 4. 证书认证
- 完成认证
- 技能徽章

## 使用示例

```bash
# 查看教程列表
openclaw tutorial list

# 开始教程
openclaw tutorial start --name "基础教程"

# 查看进度
openclaw tutorial progress

# 获取证书
openclaw tutorial certificate --name "基础教程"
```
