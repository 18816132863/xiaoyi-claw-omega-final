# ONBOARDING.md - 新手引导系统

## 目的
为零基础用户提供零门槛上手体验。

## 引导阶段

| 阶段 | 内容 | 时长 |
|------|------|------|
| 欢迎 | 欢迎体验终极鸽子王 | 10秒 |
| 能力介绍 | 核心能力介绍 | 交互式 |
| 首次交互 | 引导完成首次任务 | 交互式 |
| 模式选择 | 选择使用模式 | 交互式 |
| 完成 | 引导完成，开始使用 | - |

## 核心特性

### 1. 可跳过
- 允许跳过引导
- 随时可重新开始

### 2. 可恢复
- 进度自动保存
- 中断后可继续

### 3. 个性化
- 根据用户背景调整
- 智能推荐路径

### 4. 进度追踪
- 实时进度显示
- 完成度统计

## 使用示例

```bash
# 开始引导
openclaw onboarding start

# 跳过引导
openclaw onboarding skip

# 重新开始
openclaw onboarding restart

# 查看进度
openclaw onboarding progress
```
