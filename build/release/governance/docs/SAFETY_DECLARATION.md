## 一、能力信号说明

### 1.1 安全扫描
**用途**: 检测系统健康度、安全漏洞、配置问题

**行为说明**:
- 扫描范围：仅限用户授权的工作目录
- 扫描内容：配置文件、日志文件、系统状态
- 数据处理：本地处理，不上传敏感数据
- 用户控制：可随时禁用

### 1.2 能力信号
**用途**: 向平台报告技能能力，用于智能路由

**行为说明**:
- 信号类型：技能元数据（名称、版本、能力标签）
- 发送时机：技能加载时
- 数据范围：不包含用户数据
- 隐私保护：匿名化处理

### 1.3 加密货币
**用途**: 支持加密货币市场数据查询（可选功能）

**行为说明**:
- 功能范围：仅查询公开市场数据
- 钱包访问：需要用户明确授权
- 交易签名：需要用户二次确认
- 默认状态：禁用

## 二、本地路径访问说明

### 2.1 访问范围

| 路径类型 | 路径 | 用途 | 授权方式 |
|----------|------|------|----------|
| 工作目录 | ~/.openclaw/workspace/ | 用户文件操作 | 默认授权 |
| 配置目录 | ~/.openclaw/config/ | 系统配置读取 | 默认授权 |
| 记忆目录 | ~/.openclaw/memory/ | 记忆存储 | 默认授权 |
| 日志目录 | ~/.openclaw/logs/ | 日志记录 | 默认授权 |
| 临时目录 | /tmp/ | 临时文件处理 | 自动清理 |

### 2.2 访问原则

1. **最小权限**: 仅访问必要路径
2. **用户可控**: 用户可随时撤销授权
3. **透明可查**: 所有访问记录在日志中
4. **数据隔离**: 不同用户数据完全隔离

## 三、外部 API 访问说明

### 3.1 API 列表

| API | 用途 | 数据类型 | 授权 |
|-----|------|----------|------|
| Voyage API | 文本向量化 | 文本片段 | 默认 |
| 搜索 API | 网页搜索 | 搜索词 | 按需 |
| 天气 API | 天气查询 | 位置信息 | 按需 |
| 金融 API | 市场数据 | 股票代码 | 按需 |

### 3.2 数据保护

- **最小化原则**: 仅发送必要数据
- **脱敏处理**: 敏感信息脱敏后发送
- **加密传输**: 全程 HTTPS 加密
- **不存储**: API 响应不持久化敏感数据

## 四、自动升级机制说明

### 4.1 升级类型

| 类型 | 触发条件 | 用户确认 | 回滚能力 |
|------|----------|----------|----------|
| 配置更新 | 健康度下降 | 自动 | ✅ 支持 |
| 技能更新 | 新版本发布 | 需确认 | ✅ 支持 |
| 安全补丁 | 漏洞发现 | 自动 | ✅ 支持 |
| 架构升级 | 重大变更 | 需确认 | ✅ 支持 |

### 4.2 安全保障

1. **签名验证**: 所有升级包需签名验证
2. **完整性校验**: SHA256 校验确保完整
3. **灰度发布**: 渐进式启用，降低风险
4. **自动回滚**: 失败时自动恢复

### 4.3 用户控制

- 可禁用自动升级
- 可选择升级时机
- 可查看升级内容
- 可手动回滚
