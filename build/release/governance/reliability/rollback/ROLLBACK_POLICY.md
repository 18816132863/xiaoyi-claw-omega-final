# ROLLBACK_POLICY.md - 回滚策略

## 目的
定义系统回滚规则，确保操作失败后可恢复。

## 适用范围
- 升级失败
- 配置变更失败
- 批量操作失败
- 系统异常

## 回滚触发条件

| 触发条件 | 回滚级别 | 自动/手动 |
|----------|----------|-----------|
| 升级脚本失败 | 全量回滚 | 自动 |
| 配置写入失败 | 配置回滚 | 自动 |
| 批量操作失败 | 操作回滚 | 半自动 |
| 系统健康度<70 | 系统回滚 | 手动 |
| 用户请求回滚 | 指定回滚 | 手动 |

## 回滚点管理

### 回滚点类型
| 类型 | 创建时机 | 保留数量 |
|------|----------|----------|
| 升级前 | 每次升级前 | 5 |
| 配置变更前 | 每次配置修改 | 10 |
| 定时 | 每天0点 | 7 |
| 手动 | 用户请求 | 3 |

### 回滚点内容
```json
{
  "rollbackId": "rb_20260406_103200",
  "timestamp": "2026-04-06T10:32:00+08:00",
  "type": "pre_upgrade",
  "contents": {
    "config": "backup/config_20260406.tar.gz",
    "memory": "backup/memory_20260406.tar.gz",
    "skills": "backup/skills_20260406.tar.gz"
  },
  "checksum": "sha256:abc123...",
  "size": 10485760
}
```

### 回滚点存储
- 存储位置: `~/.openclaw/workspace/rollback/`
- 存储格式: tar.gz
- 加密: 可选AES加密
- 保留策略: 按数量轮转

## 回滚流程

### 自动回滚
```
操作失败
    ↓
检测失败类型
    ↓
选择回滚策略
    ↓
执行回滚
    ↓
验证回滚结果
    ↓
记录审计日志
    ↓
通知用户
```

### 手动回滚
```
用户请求
    ↓
验证权限
    ↓
选择回滚点
    ↓
确认回滚范围
    ↓
执行回滚
    ↓
验证回滚结果
    ↓
记录审计日志
```

## 回滚范围

### 配置回滚
| 范围 | 内容 |
|------|------|
| 核心配置 | AGENTS.md, TOOLS.md, MEMORY.md |
| 运行时配置 | runtime/*.json, runtime/*.md |
| 安全配置 | safety/*.json, safety/*.md |
| 技能配置 | skills-config.json, skills-inventory.json |

### 数据回滚
| 范围 | 内容 |
|------|------|
| 记忆数据 | memory/目录 |
| 向量索引 | ~/.openclaw/memory/main.sqlite |
| 知识图谱 | memory/ontology/ |

### 技能回滚
| 范围 | 内容 |
|------|------|
| 技能列表 | skills/目录 |
| 技能配置 | skills-config.json |
| 技能锁定 | skills-lock.json |

## 回滚验证

### 验证项
| 验证项 | 检查内容 |
|--------|----------|
| 文件完整性 | checksum校验 |
| 配置有效性 | 配置可解析 |
| 系统可用性 | 健康度检查 |
| 功能正确性 | 基本功能测试 |

### 验证失败处理
| 失败类型 | 处理 |
|----------|------|
| 文件损坏 | 尝试更早的回滚点 |
| 配置无效 | 使用默认配置 |
| 系统不可用 | 进入安全模式 |
| 功能异常 | 记录并告警 |

## 回滚命令

### 列出回滚点
```bash
openclaw rollback list
```

### 执行回滚
```bash
openclaw rollback execute --id rb_20260406_103200
```

### 创建回滚点
```bash
openclaw rollback create --type manual
```

### 验证回滚点
```bash
openclaw rollback verify --id rb_20260406_103200
```

## 异常处理

| 异常 | 处理 |
|------|------|
| 回滚点不存在 | 使用更早的回滚点 |
| 回滚失败 | 进入安全模式 |
| 验证失败 | 重试或使用其他回滚点 |
| 存储空间不足 | 清理旧回滚点 |

## 维护方式
- 调整保留数量: 修改回滚点类型配置
- 新增回滚类型: 更新回滚点类型表
- 调整验证规则: 更新验证项表
