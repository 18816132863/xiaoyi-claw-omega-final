# Infrastructure - 基础设施层

## 职责
- 工具链管理
- 技能注册表
- 路径解析
- 自动化运维

## 目录结构
```
infrastructure/
├── inventory/      # 技能注册表
├── optimization/   # 性能优化
├── fusion/         # 技能融合
├── loader/         # 加载器
└── cache/          # 缓存
```

## 核心模块
- path_resolver.py - 路径解析
- architecture_inspector.py - 架构巡检
- auto_git.py - 自动 Git
- inventory/ - 技能管理
