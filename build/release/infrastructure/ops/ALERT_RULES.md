# ALERT_RULES.md - 系统运行告警规则

## 目的
定义系统运行告警规则，确保不是出大问题才发现，而是提前预警。

## 适用范围
所有系统运行状态的监控告警。

## 告警指标

### 工具错误率异常
| 指标 | 计算方式 | 正常范围 | 告警阈值 |
|------|----------|----------|----------|
| 工具错误率 | 失败数/总调用数 | < 2% | > 5% |
| 单工具错误率 | 单工具失败数/调用数 | < 5% | > 10% |
| 连续失败次数 | 连续失败计数 | < 3 | >= 5 |

### 幻觉率上升
| 指标 | 计算方式 | 正常范围 | 告警阈值 |
|------|----------|----------|----------|
| 幻觉率 | 幻觉数/事实陈述数 | < 5% | > 10% |
| 无依据断言率 | 无依据断言/总断言 | < 3% | > 8% |
| 用户纠正率 | 用户纠正/总回答 | < 5% | > 10% |

### 平均步骤激增
| 指标 | 计算方式 | 正常范围 | 告警阈值 |
|------|----------|----------|----------|
| 平均步骤数 | 总步骤/任务数 | < 5 | > 10 |
| P95步骤数 | 95分位步骤数 | < 10 | > 20 |
| 步骤增长趋势 | 环比增长率 | < 10% | > 50% |

### 首答成功率下降
| 指标 | 计算方式 | 正常范围 | 告警阈值 |
|------|----------|----------|----------|
| 首答成功率 | 首次成功/总请求 | > 90% | < 85% |
| 追问修正率 | 需追问/总请求 | < 10% | > 20% |
| 用户满意度 | 满意评价/总评价 | > 90% | < 80% |

### 回滚频率增加
| 指标 | 计算方式 | 正常范围 | 告警阈值 |
|------|----------|----------|----------|
| 回滚频率 | 回滚次数/发布次数 | < 5% | > 20% |
| 自动回滚次数 | 自动触发回滚数 | < 1/周 | > 3/周 |
| 回滚影响范围 | 受影响用户数 | < 10 | > 100 |

### 记忆冲突激增
| 指标 | 计算方式 | 正常范围 | 告警阈值 |
|------|----------|----------|----------|
| 记忆冲突率 | 冲突数/写入数 | < 5% | > 10% |
| 无法解决冲突数 | 无法解决冲突计数 | < 1/天 | > 5/天 |
| 记忆丢失率 | 丢失数/总记忆 | < 0.1% | > 1% |

## 告警级别

| 级别 | 说明 | 通知方式 | 响应时间 |
|------|------|----------|----------|
| INFO | 信息提示 | 日志记录 | 无 |
| WARNING | 警告 | 日志 + 提示 | 24小时 |
| ERROR | 错误 | 日志 + 通知 | 4小时 |
| CRITICAL | 严重 | 日志 + 通知 + 降级 | 立即 |

## 告警触发规则

### 单指标触发
```
指标值 > 告警阈值 → 触发告警
```

### 组合触发
```
指标A > 阈值A AND 指标B > 阈值B → 触发高级别告警
```

### 趋势触发
```
指标连续3次上升 → 触发趋势告警
```

## 告警处理流程

```
告警触发
    ↓
┌─────────────────────────────────────┐
│ 1. 记录告警                          │
│    - 告警内容                        │
│    - 触发时间                        │
│    - 相关指标                        │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 发送通知                          │
│    - 按级别选择通知方式              │
│    - 发送给相关人员                  │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 自动处理（如需要）                │
│    - CRITICAL级别自动降级            │
│    - 自动恢复尝试                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 4. 人工响应                          │
│    - 问题排查                        │
│    - 修复处理                        │
│    - 记录处理结果                    │
└─────────────────────────────────────┘
```

## 告警聚合

### 聚合规则
| 规则 | 说明 |
|------|------|
| 时间窗口聚合 | 5分钟内相同告警聚合 |
| 指标关联聚合 | 相关指标告警聚合 |
| 重复告警抑制 | 重复告警抑制 |

### 聚合格式
```json
{
  "alertGroupId": "ag_001",
  "alerts": [
    {
      "alertId": "alert_001",
      "metric": "tool_error_rate",
      "value": 0.08
    }
  ],
  "aggregatedAt": "2026-04-06T10:32:00+08:00"
}
```

## 异常处理

| 异常 | 处理 |
|------|------|
| 告警发送失败 | 重试 + 备用通道 |
| 告警风暴 | 聚合 + 抑制 |
| 响应超时 | 升级通知 |

## 维护方式
- 新增告警指标: 添加到告警指标表
- 调整告警阈值: 更新阈值配置
- 新增告警级别: 更新级别表
