# CACHE_POLICY.md - 缓存策略

## 目的
定义可缓存内容、缓存时机、失效条件，确保系统能在不破坏正确性的前提下降成本。

## 适用范围
所有可缓存内容的缓存管理。

## 可缓存内容

### 检索结果缓存
| 内容类型 | 可缓存 | 缓存时长 | 说明 |
|----------|--------|----------|------|
| 稳定知识 | ✅ | 24小时 | 百科、定义等 |
| 一般网页 | ✅ | 1小时 | 普通网页内容 |
| 新闻资讯 | ⚠️ | 15分钟 | 时效敏感 |
| 实时数据 | ❌ | 0 | 价格、天气等 |

### 查询改写缓存
| 内容类型 | 可缓存 | 缓存时长 | 说明 |
|----------|--------|----------|------|
| 改写结果 | ✅ | 1小时 | 相同查询可复用 |
| 实体识别 | ✅ | 2小时 | 实体识别结果 |
| 意图分析 | ✅ | 30分钟 | 意图分析结果 |

### 会话摘要缓存
| 内容类型 | 可缓存 | 缓存时长 | 说明 |
|----------|--------|----------|------|
| 会话摘要 | ✅ | 会话期间 | 当前会话有效 |
| 目标提取 | ✅ | 会话期间 | 当前会话有效 |
| 约束提取 | ✅ | 会话期间 | 当前会话有效 |

### 低风险工具返回缓存
| 内容类型 | 可缓存 | 缓存时长 | 说明 |
|----------|--------|----------|------|
| 文件读取 | ✅ | 5分钟 | 文件未修改时 |
| 记忆搜索 | ✅ | 10分钟 | 记忆未更新时 |
| 配置读取 | ✅ | 30分钟 | 配置未修改时 |

## 禁止缓存内容

### 高敏感内容
| 内容类型 | 原因 |
|----------|------|
| 密码 | 安全风险 |
| API密钥 | 安全风险 |
| 个人隐私 | 隐私保护 |
| 认证Token | 安全风险 |

### 强时效内容
| 内容类型 | 原因 |
|----------|------|
| 实时价格 | 时效性 |
| 天气预报 | 时效性 |
| 新闻动态 | 时效性 |
| 股票行情 | 时效性 |

### 高冲突风险内容
| 内容类型 | 原因 |
|----------|------|
| 用户画像 | 可能变化 |
| 偏好设置 | 可能变化 |
| 决策记录 | 可能更新 |
| 状态信息 | 可能变化 |

## 缓存失效条件

### 时间失效
| 条件 | 说明 |
|------|------|
| TTL到期 | 缓存时间到期 |
| 会话结束 | 会话相关缓存失效 |
| 系统重启 | 内存缓存失效 |

### 事件失效
| 条件 | 说明 |
|------|------|
| 数据更新 | 原数据已更新 |
| 配置变更 | 相关配置已变更 |
| 用户操作 | 用户触发失效 |

### 主动失效
| 条件 | 说明 |
|------|------|
| 检测到过期 | 检测到内容过期 |
| 冲突检测 | 检测到信息冲突 |
| 质量下降 | 缓存质量下降 |

## 缓存策略

### 缓存层级
| 层级 | 说明 | 存储位置 |
|------|------|----------|
| L1 | 内存缓存 | 进程内存 |
| L2 | 持久缓存 | 本地文件 |
| L3 | 分布式缓存 | Redis等 |

### 缓存策略
| 策略 | 说明 |
|------|------|
| LRU | 最近最少使用 |
| LFU | 最少频率使用 |
| TTL | 时间到期失效 |

### 缓存大小限制
| 层级 | 最大大小 | 淘汰策略 |
|------|----------|----------|
| L1 | 100MB | LRU |
| L2 | 1GB | LRU + TTL |
| L3 | 10GB | LFU + TTL |

## 缓存命中处理

### 命中时
| 处理 | 说明 |
|------|------|
| 返回缓存 | 直接返回缓存内容 |
| 更新统计 | 更新命中统计 |
| 检查新鲜度 | 可选检查内容新鲜度 |

### 未命中时
| 处理 | 说明 |
|------|------|
| 执行查询 | 执行实际查询 |
| 存储缓存 | 存储结果到缓存 |
| 返回结果 | 返回查询结果 |

## 缓存统计

### 统计指标
| 指标 | 说明 |
|------|------|
| hit_rate | 命中率 |
| miss_rate | 未命中率 |
| avg_latency | 平均延迟 |
| cache_size | 缓存大小 |

### 统计格式
```json
{
  "timestamp": "2026-04-06T10:32:00+08:00",
  "stats": {
    "hit_rate": 0.75,
    "miss_rate": 0.25,
    "avg_latency_ms": 50,
    "cache_size_mb": 45,
    "entries": 1500
  }
}
```

## 异常处理

| 异常 | 处理 |
|------|------|
| 缓存读取失败 | 重新查询 |
| 缓存写入失败 | 跳过缓存 |
| 缓存过期 | 重新查询 |

## 维护方式
- 新增可缓存类型: 添加到可缓存表
- 调整缓存时长: 更新缓存时长配置
- 新增失效条件: 更新失效条件表
