# CONTEXT_WINDOW_MANAGE.md - 上下文窗口管理

## 目的
定义上下文窗口的管理策略，确保有效利用有限的上下文空间。

## 适用范围
所有 LLM 调用的上下文构建和管理。

## 上下文窗口分配

### 标准分配
| 组件 | 分配比例 | 说明 |
|------|----------|------|
| 系统提示 | 10% | 核心系统指令 |
| 用户画像 | 5% | 用户偏好和约束 |
| 技能定义 | 15% | 当前任务相关技能 |
| 会话历史 | 30% | 对话上下文 |
| 检索内容 | 25% | 外部检索结果 |
| 输出预留 | 15% | 输出空间预留 |

### 动态调整
```yaml
dynamic_allocation:
  by_task_type:
    qa:
      history: 40%
      retrieval: 30%
    document:
      retrieval: 40%
      history: 20%
    multi_step:
      skills: 25%
      history: 35%
```

## 上下文构建流程

```
任务开始
    ↓
┌─────────────────────────────────────┐
│ 1. 计算可用窗口                      │
│    - 获取模型窗口大小                │
│    - 减去输出预留                    │
│    - 计算各组件配额                  │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 加载核心组件                      │
│    - 系统提示 (固定)                 │
│    - 用户画像 (固定)                 │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 加载动态组件                      │
│    - 技能定义 (按优先级)             │
│    - 会话历史 (压缩)                 │
│    - 检索内容 (筛选)                 │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 4. 检查窗口限制                      │
│    - 超限 → 压缩/截断               │
│    - 未超限 → 使用                  │
└─────────────────────────────────────┘
    ↓
构建完成
```

## 历史压缩策略

### 压缩方法
| 方法 | 说明 | 压缩率 | 适用场景 |
|------|------|--------|----------|
| 摘要压缩 | 生成历史摘要 | 60-80% | 长对话 |
| 滑动窗口 | 保留最近N轮 | 50-70% | 标准对话 |
| 关键提取 | 提取关键信息 | 70-90% | 信息密集 |
| 分层压缩 | 按重要性分层 | 40-60% | 复杂任务 |

### 压缩规则
```yaml
history_compression:
  triggers:
    - window_usage > 80%
    - history_turns > 20
  
  strategy: summary
  
  summary_config:
    max_length: 500
    preserve_key_facts: true
    preserve_user_constraints: true
    preserve_pending_tasks: true
```

## 技能加载优化

### 技能选择
| 规则 | 说明 |
|------|------|
| 相关性优先 | 只加载任务相关技能 |
| 大小限制 | 单技能最大 2KB |
| 总量限制 | 技能总大小 < 10KB |
| 优先级排序 | P0 > P1 > P2 > P3 |

### 技能精简
```yaml
skill_trimming:
  remove_sections:
    - examples          # 移除示例
    - references        # 移除参考
    - detailed_docs     # 移除详细文档
  
  keep_sections:
    - purpose           # 保留目的
    - usage             # 保留用法
    - constraints       # 保留约束
```

## 检索内容筛选

### 筛选规则
| 规则 | 说明 |
|------|------|
| 相关性筛选 | 只保留高相关内容 |
| 时效性筛选 | 优先最新内容 |
| 来源筛选 | 优先可信来源 |
| 去重筛选 | 移除重复内容 |

### 内容限制
```yaml
retrieval_limits:
  max_items: 10
  max_item_length: 500
  max_total_length: 5000
  min_relevance: 0.7
```

## 窗口监控

### 监控指标
| 指标 | 说明 | 告警阈值 |
|------|------|----------|
| 窗口使用率 | 已用/总量 | > 90% |
| 压缩触发率 | 压缩次数/请求 | > 30% |
| 截断率 | 截断次数/请求 | > 10% |
| 信息丢失率 | 丢失关键信息 | > 5% |

## 异常处理

| 异常 | 处理 |
|------|------|
| 窗口超限 | 压缩历史 + 精简技能 |
| 压缩失败 | 截断最旧历史 |
| 关键信息丢失 | 标记 + 请求用户确认 |

## 维护方式
- 调整分配: 修改分配比例
- 新增策略: 添加压缩策略
- 调整限制: 更新限制配置
