# LAZY_LOADING.md - 延迟加载策略

## 目的
定义技能、上下文、资源的延迟加载策略，优化启动性能和资源使用。

## 适用范围
所有技能加载、上下文构建、资源初始化。

## 延迟加载原则

### 核心原则
1. **按需加载**: 只在需要时加载
2. **预加载关键**: 关键资源提前加载
3. **缓存复用**: 已加载资源缓存复用
4. **异步加载**: 非阻塞异步加载

## 技能延迟加载

### 加载优先级
| 优先级 | 加载策略 | 技能类型 |
|--------|----------|----------|
| P0 | 立即加载 | xiaoyi 系列、核心技能 |
| P1 | 预加载 | 搜索、文档处理 |
| P2 | 按需加载 | 开发、分析工具 |
| P3 | 延迟加载 | 辅助、工具类 |

### 加载触发
```yaml
load_triggers:
  P0:
    - system_start       # 系统启动
    - always             # 始终保持
  
  P1:
    - intent_match       # 意图匹配
    - user_request       # 用户请求
    - preload_hint       # 预加载提示
  
  P2:
    - explicit_call      # 显式调用
    - skill_chain        # 技能链调用
    - fallback           # 备选调用
  
  P3:
    - explicit_call      # 仅显式调用
    - manual_load        # 手动加载
```

### 加载流程
```
技能请求
    ↓
┌─────────────────────────────────────┐
│ 1. 检查缓存                          │
│    - 已缓存 → 直接使用               │
│    - 未缓存 → 继续加载               │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. 加载技能                          │
│    - 读取 SKILL.md                   │
│    - 解析技能配置                    │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. 缓存技能                          │
│    - 存入技能缓存                    │
│    - 设置过期时间                    │
└─────────────────────────────────────┘
    ↓
使用技能
```

## 上下文延迟加载

### 上下文分层
| 层级 | 加载策略 | 内容 |
|------|----------|------|
| 核心 | 立即加载 | 系统提示、用户画像 |
| 会话 | 按需加载 | 会话历史、上下文 |
| 技能 | 延迟加载 | 技能说明、示例 |
| 参考 | 按需加载 | 参考文档、知识库 |

### 上下文压缩
```yaml
context_compression:
  history:
    strategy: summary      # 摘要压缩
    max_turns: 10          # 保留最近10轮
    summary_ratio: 0.3     # 摘要压缩比
  
  skills:
    strategy: essential    # 只加载必要
    max_size_per_skill: 2048
    total_limit: 10000
```

## 资源延迟加载

### 资源类型
| 资源 | 加载策略 | 缓存时间 |
|------|----------|----------|
| 配置文件 | 启动加载 | 永久 |
| 模型 | 按需加载 | 会话内 |
| 向量索引 | 按需加载 | 1小时 |
| 外部数据 | 按需加载 | 请求内 |

### 资源池管理
```yaml
resource_pool:
  max_size: 100MB
  eviction_policy: LRU
  preload:
    - system_config
    - user_profile
  lazy_load:
    - skill_definitions
    - knowledge_base
    - external_apis
```

## 预加载策略

### 预加载触发
| 触发条件 | 预加载内容 |
|----------|------------|
| 意图识别完成 | 相关技能 |
| 任务类型确定 | 相关资源 |
| 用户习惯匹配 | 常用技能 |
| 时间规律匹配 | 定时任务 |

### 预加载限制
```yaml
preload_limits:
  max_concurrent: 3       # 最大并发预加载
  max_memory: 20MB        # 最大预加载内存
  timeout: 5000           # 预加载超时
  on_failure: skip        # 失败时跳过
```

## 缓存管理

### 缓存层级
| 层级 | 内容 | 大小限制 | TTL |
|------|------|----------|-----|
| L1 热缓存 | 当前会话技能 | 10MB | 会话内 |
| L2 温缓存 | 常用技能 | 50MB | 1小时 |
| L3 冷缓存 | 所有已加载 | 100MB | 1天 |

### 缓存策略
```yaml
cache_policy:
  eviction: LRU
  hit_target: 0.8
  refresh_on_access: true
  background_refresh: true
```

## 性能指标

### 加载性能
| 指标 | 目标 | 告警阈值 |
|------|------|----------|
| P0 加载时间 | < 100ms | > 200ms |
| P1 加载时间 | < 500ms | > 1s |
| P2 加载时间 | < 1s | > 2s |
| 缓存命中率 | > 80% | < 60% |

### 资源使用
| 指标 | 目标 | 告警阈值 |
|------|------|----------|
| 内存使用 | < 100MB | > 150MB |
| 预加载内存 | < 20MB | > 30MB |
| 缓存内存 | < 50MB | > 80MB |

## 异常处理

| 异常 | 处理 |
|------|------|
| 加载超时 | 使用缓存或降级 |
| 内存不足 | 清理 L3 缓存 |
| 加载失败 | 记录日志，使用备选 |

## 维护方式
- 调整优先级: 更新优先级配置
- 调整缓存: 更新缓存配置
- 新增触发: 更新触发条件
