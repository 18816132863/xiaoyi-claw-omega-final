## 使用场景

### 场景1: 复杂任务处理
```
用户请求 → 推理分析 → 决策选择 → 规划分解 → 技能执行 → 结果审查 → 记忆存储
```

### 场景2: 故障恢复
```
故障检测 → 风险评估 → 恢复策略 → 自动恢复 → 结果验证 → 审计记录
```

### 场景3: 自动化运维
```
定时触发 → 任务调度 → 资源分配 → 并发执行 → 结果收集 → 告警通知
```
