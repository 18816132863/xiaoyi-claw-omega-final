### 2.1 PolicyEngine.evaluate_policy() - 策略评估统一入口

```python
def evaluate_policy(
    task_meta: Dict,
    profile: str,
    requested_capabilities: List[str] = None
) -> Dict:
    """
    ═══════════════════════════════════════════════════════════════
    架构统一入口 - 所有请求必须经过此接口
    ═══════════════════════════════════════════════════════════════
    
    调用链：
    ┌─────────────────────────────────────────────────────────────┐
    │ 1. _load_metrics()                                          │
    │    └── 读取 reports/metrics/aggregated_metrics.json         │
    │                                                              │
    │ 2. RiskClassifier.classify(task_meta, profile)              │
    │    └── 返回 RiskAssessment { level, score, factors }        │
    │                                                              │
    │ 3. PermissionEngine.check_permissions(profile, ...)         │
    │    └── 返回 PermissionCheck { allowed, granted, denied }    │
    │                                                              │
    │ 4. TokenBudgetManager.get_decision_budget(profile)          │
    │    └── 返回 { remaining, used, limit }                      │
    │                                                              │
    │ 5. CostBudgetManager.get_decision_budget(profile)           │
    │    └── 返回 { api_call, compute, storage, network }         │
    │                                                              │
    │ 6. HighRiskGuard.guard(assessment, profile, ...)            │
    │    └── 返回 GuardDecision { action, reason }                │
    │                                                              │
    │ 7. _adjust_by_metrics(metrics, profile)                     │
    │    └── 基于 task_success_rate, skill_failure_rate 调整      │
    │       └── 低成功率 → degrade                                │
    │       └── 高失败率 → degrade                                │
    │       └── poor health → degrade                             │
    └─────────────────────────────────────────────────────────────┘
    
    返回决策对象：
    ┌─────────────────────────────────────────────────────────────┐
    │ {                                                           │
    │   "decision": "allow" | "deny" | "degrade" | "review",      │
    │   "allowed": bool,                                          │
    │   "risk_level": "low" | "medium" | "high" | "critical",     │
    │   "risk_score": float,                                      │
    │   "token_budget": int,                                      │
    │   "cost_budget": float,                                     │
    │   "allowed_capabilities": list,                             │
    │   "blocked_capabilities": list,                             │
    │   "requires_review": bool,                                  │
    │   "reason": str,                                            │
    │   "mitigations": list,                                      │
    │   "metrics": {                                              │
    │     "task_success_rate": float,                             │
    │     "skill_failure_rate": float,                            │
    │     "memory_hit_rate": float,                               │
    │     "overall_health": str                                   │
    │   }                                                         │
    │ }                                                           │
    └─────────────────────────────────────────────────────────────┘
    """
```

### 2.2 SkillRouter.select_skill() - 技能选择统一入口

```python
def select_skill(
    intent: str,
    profile: str,
    constraints: Dict = None,
    governance_decision: Dict = None
) -> Dict:
    """
    ═══════════════════════════════════════════════════════════════
    架构统一入口 - 技能选择必须经过此接口
    ═══════════════════════════════════════════════════════════════
    
    调用链：
    ┌─────────────────────────────────────────────────────────────┐
    │ 1. _check_metrics_reload()                                  │
    │    └── 检查 reports/metrics/skill_metrics.json mtime        │
    │    └── 如果更新则重新加载                                    │
    │                                                              │
    │ 2. registry.reload()                                        │
    │    └── 从 skills/registry/skill_registry.json 重新加载      │
    │                                                              │
    │ 3. 按 intent 搜索候选技能                                    │
    │    └── registry.search(intent)                              │
    │                                                              │
    │ 4. 按 profile 过滤分类                                       │
    │    └── admin: ["*"]                                          │
    │    └── developer: ["code", "git", "docker", ...]            │
    │    └── default: ["utility", "search", "document"]           │
    │                                                              │
    │ 5. 按 status 过滤                                            │
    │    └── 排除 DISABLED, DEPRECATED                            │
    │    └── 保留 STABLE, EXPERIMENTAL                            │
    │                                                              │
    │ 6. 按 tags 过滤                                              │
    │    └── constraints.get("required_tags", [])                 │
    │                                                              │
    │ 7. 按 governance_decision 过滤                               │
    │    └── 排除 blocked_capabilities 中的技能                    │
    │                                                              │
    │ 8. _sort_by_metrics(candidates)                             │
    │    └── 按 failure_rate 升序（低失败率优先）                  │
    │    └── 按 avg_latency_ms 升序（低延迟优先）                  │
    │    └── 综合评分 = failure_rate * 100 + latency / 1000       │
    └─────────────────────────────────────────────────────────────┘
    
    返回选择结果：
    ┌─────────────────────────────────────────────────────────────┐
    │ {                                                           │
    │   "success": bool,                                          │
    │   "skill_id": str,                                          │
    │   "confidence": float,                                      │
    │   "reason": str,                                            │
    │   "profile": str,                                           │
    │   "category": str,                                          │
    │   "status": str,                                            │
    │   "tags": list                                              │
    │ }                                                           │
    └─────────────────────────────────────────────────────────────┘
    """
```

### 2.3 WorkflowEngine.run_workflow() - 工作流执行统一入口

```python
def run_workflow(
    workflow_spec: Dict,
    profile: str = "default",
    context_bundle: Dict = None,
    resume_from_checkpoint: str = None
) -> WorkflowResult:
    """
    ═══════════════════════════════════════════════════════════════
    架构统一入口 - 工作流执行必须经过此接口
    ═══════════════════════════════════════════════════════════════
    
    注入组件：
    ┌─────────────────────────────────────────────────────────────┐
    │ self.fallback_policy = FallbackPolicy()                     │
    │ self.rollback_manager = RollbackManager()                   │
    │ self.checkpoint_store = CheckpointStore()                   │
    └─────────────────────────────────────────────────────────────┘
    
    执行流程：
    ┌─────────────────────────────────────────────────────────────┐
    │ 1. _build_dag(steps)                                        │
    │    └── 构建依赖图 { nodes, edges, entry_points, exit_points }│
    │                                                              │
    │ 2. 加载 checkpoint（如果 resume_from_checkpoint）            │
    │    └── checkpoint_store.load(checkpoint_id)                 │
    │                                                              │
    │ 3. _execute_dag(dag, result, context)                       │
    │    ┌─────────────────────────────────────────────────────┐  │
    │    │ for each step:                                       │  │
    │    │   ├── checkpoint_store.save() # step 前              │  │
    │    │   ├── _execute_step()                                │  │
    │    │   │   ├── _execute_action()                          │  │
    │    │   │   │   └── skill_router.execute()                 │  │
    │    │   │   │       └── 失败时抛出 skill_router_error:{e}  │  │
    │    │   │   └── 失败时 FallbackPolicy.decide()             │  │
    │    │   │       ├── RETRY → 重试                           │  │
    │    │   │       ├── FALLBACK → 执行 fallback_skill         │  │
    │    │   │       ├── SKIP → 跳过                             │  │
    │    │   │       └── ABORT → 中止                            │  │
    │    │   └── checkpoint_store.save() # step 后              │  │
    │    └─────────────────────────────────────────────────────┘  │
    │                                                              │
    │ 4. 保存最终 checkpoint                                       │
    │    └── checkpoint_store.save(workflow_id, "_completed", ...)│
    └─────────────────────────────────────────────────────────────┘
    
    返回结果：
    ┌─────────────────────────────────────────────────────────────┐
    │ WorkflowResult:                                              │
    │   - workflow_id: str                                        │
    │   - status: "completed" | "failed" | ...                    │
    │   - step_results: Dict[str, StepResult]                     │
    │   - failed_step_id: str                                     │
    │   - reason: str                                             │
    │   - total_retry_count: int                                  │
    │   - fallback_used: bool                                     │
    │   - rollback_used: bool                                     │
    │   - checkpoint_id: str                                      │
    │   - total_duration_ms: int                                  │
    └─────────────────────────────────────────────────────────────┘
    """
```
