# 抖店智能客服系统 - 功能说明书 V1.0

## 一、系统概述

### 1.1 系统定位
基于华为云部署的抖店智能客服系统，实现自动回复、订单处理、售后管理等功能。

### 1.2 技术架构
```
抖店 API → 华为云 API 网关 → 业务逻辑层 → 数据库
                ↓
           智能回复引擎
```

### 1.3 部署环境
- **云服务商**: 华为云
- **服务器**: ECS 4核8G
- **数据库**: RDS MySQL 2核4G
- **缓存**: DCS Redis 2G
- **存储**: OBS 对象存储

## 四、系统流程

### 4.1 客服消息处理流程

```
用户消息 
    ↓
飞鸽推送 
    ↓
华为云 API 网关
    ↓
意图识别
    ├── 商品咨询 → 自动回复商品信息
    ├── 订单查询 → 查询订单 → 返回状态
    ├── 售后申请 → 创建工单 → 自动处理/转人工
    └── 投诉建议 → 转人工客服
```

### 4.2 订单自动处理流程

```
新订单创建
    ↓
系统监听（Webhook）
    ↓
订单状态判断
    ├── 待付款 → 30分钟后催付
    ├── 已付款 → 通知发货
    ├── 已发货 → 物流跟踪
    └── 已签收 → 邀请评价
```

### 4.3 售后处理流程

```
售后申请
    ↓
风险评估
    ├── 低风险 → 自动同意退款
    ├── 中风险 → 人工审核
    └── 高风险 → 纠纷预警
    ↓
处理结果通知
```

## 五、配置项

### 5.1 自动回复规则

| 规则类型 | 关键词 | 回复内容 |
|----------|--------|----------|
| 关键词 | 发货 | "亲，订单付款后48小时内发货哦~" |
| 关键词 | 退款 | "亲，如需退款请在订单详情页申请~" |
| 正则 | 订单号:\d+ | 自动查询订单状态 |
| 意图 | 投诉 | 转人工客服 |

### 5.2 工作时间配置

| 时段 | 处理方式 | 响应时间 |
|------|----------|----------|
| 9:00-22:00 | 自动回复 + 人工 | < 30秒 |
| 22:00-9:00 | 仅自动回复 | < 5秒 |

### 5.3 告警规则

| 事件 | 阈值 | 告警方式 |
|------|------|----------|
| API 调用失败 | 失败率 > 5% | 短信 + 邮件 |
| 响应超时 | > 10秒 | 钉钉通知 |
| 纠纷预警 | 高风险订单 | 电话通知 |
| 系统异常 | 服务不可用 | 电话 + 短信 |
