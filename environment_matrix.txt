------------------------------------------------------------
  一、环境类型
------------------------------------------------------------

| 环境 | 数据库 | Redis | 用途 |
|------|--------|-------|------|
| local | SQLite | 否 | 本地开发 |
| test | SQLite | Mock | 测试环境 |
| prod | PostgreSQL | 是 | 生产环境 |

------------------------------------------------------------
  二、配置文件
------------------------------------------------------------

| 文件 | 说明 |
|------|------|
| .env.example | 通用示例 |
| .env.local.example | 本地环境示例 |
| .env.prod.example | 生产环境示例 |

------------------------------------------------------------
  三、环境变量
------------------------------------------------------------

| 变量 | local | test | prod |
|------|-------|------|------|
| OPENCLAW_ENV | local | test | prod |
| OPENCLAW_DEBUG | true | false | false |
| DATABASE_URL | sqlite | sqlite | postgresql |
| REDIS_HOST | 空 | 空 | redis |
| MESSAGE_SERVER_PORT | 18790 | 18790 | 18790 |
| DAEMON_CHECK_INTERVAL | 5.0 | 5.0 | 5.0 |

------------------------------------------------------------
  四、配置模块
------------------------------------------------------------

位置: infrastructure/config/

类:
- Settings: 全局配置
- DatabaseConfig: 数据库配置
- RedisConfig: Redis 配置
- MessageServerConfig: 消息服务配置
- DaemonConfig: 守护进程配置
- ObservabilityConfig: 可观测性配置

使用:
```python
from infrastructure.config import get_settings
settings = get_settings()
```

------------------------------------------------------------
  五、主链集成
------------------------------------------------------------

已集成组件:
- scripts/task_daemon.py: 从 settings 读取配置
- scripts/message_server.py: 从 settings 读取配置
- scripts/start_services.sh: 支持环境变量
