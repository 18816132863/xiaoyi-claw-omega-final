## 🔄 工作流

### 电商选品分析
```
帮我分析一下淘宝上'蓝牙耳机'的选品趋势
```
**步骤**: 搜索 → 分析价格/销量 → 生成报告

### 工厂筛选比价
```
帮我找几家做手机壳的工厂，对比一下价格
```
**步骤**: 搜索供应商 → 收集报价 → 对比分析

### 主播/团长筛选
```
帮我找一些适合推广美妆产品的主播
```
**步骤**: 搜索主播 → 评估数据 → 生成推荐

### 店铺启动
```
我要开一家淘宝店，帮我规划启动流程
```
**步骤**: 确定定位 → 规划产品 → 制定计划

### 代码审计
```
帮我审计一下这个项目的代码质量
```
**步骤**: 扫描结构 → 检查规范 → 生成报告

## 💡 使用技巧

### 1. 自然语言
直接描述需求，无需记忆命令：
```
帮我分析一下最近的咖啡市场趋势，生成一份报告
```

### 2. 多步骤任务
我会自动拆解复杂任务：
```
帮我找几家做包装的工厂，对比价格和质量，然后生成一份推荐报告
```

### 3. 记忆功能
让我记住重要信息：
```
记住我的店铺名称是"星空小店"，主营类目是"数码配件"
```

### 4. 定时任务
设置周期性任务：
```
每天早上9点提醒我查看昨天的销售数据
每周一提醒我复盘上周工作
```

### 5. 文件操作
读取和创建文件：
```
读取一下销售数据.xlsx
把这些分析结果生成一份Word报告
```

## ⚠️ 注意事项

1. **网络访问**: 部分功能需要网络连接
2. **文件格式**: 支持常见格式（docx, pdf, xlsx, csv等）
3. **定时任务**: 需要明确时间格式
4. **敏感信息**: 不会主动读取或泄露敏感数据
