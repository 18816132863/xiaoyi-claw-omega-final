# L2 Memory Context Layer - 升级版

记忆上下文层核心模块，包含多模态搜索、跨语言搜索、智能维护等高级功能。

## 模块列表

### multimodal/ - 多模态搜索系统
- `multimodal_search.py` - 图像/音频向量搜索、CLIP模型集成

### cross_lingual/ - 跨语言搜索系统
- `cross_lingual.py` - 多语言支持、语言检测、跨语言向量对齐

### maintenance/ - 智能维护系统
- `run_maintenance.py` - 完整维护任务
- `check_coverage.py` - 覆盖率检查
- `vector_system_optimizer.py` - 向量系统优化
