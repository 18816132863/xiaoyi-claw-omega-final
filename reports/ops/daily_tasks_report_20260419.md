- 错过了 00:00-09:00 的所有任务

- 心跳每 15 分钟执行一次

- 如果心跳时间不在窗口内，任务会被跳过

## 💡 改进建议

### 1. 扩大触发窗口
将触发窗口从 30 分钟扩大到 60 分钟：

```python
# 当前
if current_hour == 9 and current_minute < 30:

# 改为
if current_hour == 9:
```

### 2. 添加补执行机制
如果任务当天未执行，在下次心跳时补执行：

```python
# 检查任务是否已执行
if "daily_health_reminder" not in today_triggered:
    # 即使不在窗口内也执行
    triggers.append(...)
```

### 3. 启动时检查所有任务
守护进程启动时，检查今天所有应该执行但未执行的任务：

```python
# 启动时检查
for task in daily_tasks:
    if task.time < current_time and task not in executed:
        execute(task)
```
