# 中文文案模板库

## 一、短视频脚本模板

### 1. 产品种草类

```markdown
【开场】(3秒)
- 画面: 产品特写
- 文案: "这个[产品名]真的绝了！"

【痛点】(5秒)
- 画面: 使用前对比
- 文案: "你是不是也遇到过[痛点]？"

【解决方案】(10秒)
- 画面: 产品使用过程
- 文案: "用了这个之后，[效果描述]"

【信任背书】(5秒)
- 画面: 用户评价/数据
- 文案: "已经有[X]万人都在用"

【行动号召】(3秒)
- 画面: 购买链接/优惠信息
- 文案: "现在下单还有限时优惠，点击下方链接"
```

### 2. 知识科普类

```markdown
【悬念开场】(3秒)
- 文案: "你知道吗？[反常识观点]"

【引入主题】(5秒)
- 文案: "今天我们来聊聊[主题]"

【核心内容】(15秒)
- 要点1: [内容]
- 要点2: [内容]
- 要点3: [内容]

【总结升华】(5秒)
- 文案: "所以记住，[核心观点]"

【互动引导】(3秒)
- 文案: "你还有什么想了解的？评论区告诉我"
```

### 3. 情感共鸣类

```markdown
【场景切入】(5秒)
- 画面: 生活场景
- 文案: "有没有人和我一样，[共鸣点]"

【情感铺垫】(10秒)
- 画面: 情感画面
- 文案: "[情感描述]"

【转折升华】(5秒)
- 画面: 正能量画面
- 文案: "但是后来我发现，[感悟]"

【行动号召】(3秒)
- 文案: "如果你也有同感，点个赞吧"
```

## 三、多平台适配

### 抖音
- 时长: 15-60秒
- 风格: 快节奏、强情绪
- 文案: 口语化、有梗

### 小红书
- 时长: 图文为主
- 风格: 精致、种草
- 文案: 真实体验、细节丰富

### 公众号
- 时长: 长文
- 风格: 专业、深度
- 文案: 结构清晰、有逻辑

### B站
- 时长: 3-15分钟
- 风格: 硬核、干货
- 文案: 专业术语、有梗

## 五、使用方法

```python
# 调用示例
from copywriter import Copywriter

# 生成短视频脚本
script = Copywriter.generate_short_video(
    type="产品种草",
    product="智能水杯",
    pain_point="总是忘记喝水",
    effect="每天自动提醒，再也不怕缺水"
)

# 生成营销文案
copy = Copywriter.generate_marketing(
    platform="小红书",
    product="护肤精华",
    target="25-35岁女性",
    selling_points=["补水", "抗衰", "提亮"]
)

# 生成A/B测试变体
variants = Copywriter.generate_ab_variants(
    original="如何提高工作效率？",
    count=3
)
```
