# daily-ai-news

搜索过去24小时 AI 领域重要新闻，按分类整理后发送到佳音私聊。

## 执行步骤

1. 使用 web-search 技能搜索过去24小时 AI 领域重要新闻
2. 按分类整理，去重，筛选最重要的不超过10条
3. 验证每条链接真实可访问（不得虚构链接）
4. 按输出格式发送到佳音私聊（`user:ou_6a42e55836c49451e235e83dee4a6c8f`）

## 信息来源（优先级顺序）

36氪、华尔街见闻、新华社、人民日报、TechCrunch、Reuters、Axios、SemiAnalysis、The Information、Bloomberg

不足时可补充：其他可信媒体、公司官网、官方博客、监管机构公告

## 分类（可灵活调整）

- 宏观趋势
- 出圈产品
- 技术迭代
- 名人动态
- 政策法规

## 输出格式

直接输出内容，无其他说明文字：

```
**{分类名}**
• [{标题}]({链接）
  {一句话摘要，不超过50字}

**{分类名}**
• ...
```

**规则：**
- 全程中文
- 每条必须包含标题、链接、摘要
- 链接必须是真实可打开的，禁止虚构
- 总条数不超过10条
- 无新闻的分类不显示
