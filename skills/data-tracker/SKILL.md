# 数据记录与复盘系统

记录你关心的数据，每天自动复盘，帮你养成好习惯。

## 快速参考

| 场景 | 操作 |
|------|------|
| 记录数据 | `记录今天：睡眠7小时，运动30分钟` |
| 查看今日 | `今天记录了什么` |
| 每日复盘 | `生成今日复盘` |
| 周报汇总 | `生成本周报告` |
| 趋势分析 | `分析最近7天睡眠趋势` |

---

## 数据类型

### 默认支持

| 类型 | 单位 | 示例 |
|------|------|------|
| 睡眠 | 小时 | `睡眠7小时` |
| 运动 | 分钟 | `运动30分钟` |
| 阅读 | 页/分钟 | `阅读20页` |
| 工作 | 小时 | `工作8小时` |
| 喝水 | 杯/ml | `喝水8杯` |
| 步数 | 步 | `步数10000步` |
| 体重 | kg | `体重65kg` |
| 心情 | 1-10 | `心情8分` |

### 自定义数据

用户可以定义任意数据类型：
```
添加数据类型：写作字数，单位：字
记录今天：写作2000字
```

## 复盘报告

### 每日复盘

```markdown
# 2026-04-17 复盘

## 数据汇总
- 睡眠: 7小时 (目标: 8小时) ⚠️
- 运动: 30分钟 (目标: 30分钟) ✅
- 阅读: 20页 (目标: 30页) ⚠️
- 喝水: 8杯 (目标: 8杯) ✅

## 完成率
- 总体: 75%
- 达标项: 2/4

## 建议
- 睡眠不足1小时，建议今晚早点休息
- 阅读进度落后，明天可以多读10页

## 最佳决策
- 坚持运动30分钟

## 明天计划
- 睡眠: 目标8小时
- 运动: 继续30分钟
- 阅读: 补上今天落后的10页
```

### 每周报告

```markdown
# 2026年第16周报告

## 数据趋势
- 平均睡眠: 6.8小时 (↓0.5)
- 平均运动: 28分钟 (↑3)
- 总阅读: 150页 (↑20)

## 习惯完成率
- 睡眠达标: 5/7 (71%)
- 运动达标: 6/7 (86%)
- 阅读达标: 4/7 (57%)

## 本周亮点
- 运动坚持得很好
- 周三睡眠达到8小时

## 需要改进
- 周末睡眠不足
- 阅读进度不稳定

## 下周目标
- 睡眠达标率提升到80%
- 阅读每天至少30页
```

## 与长期记忆集成

### 沉淀规则

只沉淀高价值信息：

| 沉淀类型 | 条件 | 示例 |
|----------|------|------|
| 习惯养成 | 连续7天达标 | "用户习惯每天运动30分钟" |
| 目标调整 | 多次未达标 | "用户睡眠目标可能需要调整" |
| 最佳实践 | 发现有效方法 | "用户发现早起运动效果更好" |
| 异常预警 | 数据异常 | "用户连续3天睡眠不足6小时" |

### 不沉淀

- 临时噪音数据
- 单次记录
- 无规律数据

## 配置

### 目标设置

```json
{
  "goals": {
    "sleep": {"target": 8, "unit": "hours"},
    "exercise": {"target": 30, "unit": "minutes"},
    "reading": {"target": 30, "unit": "pages"},
    "water": {"target": 8, "unit": "cups"}
  }
}
```

### 提醒设置

```json
{
  "reminders": {
    "morning": "09:00 - 提醒记录昨晚睡眠",
    "evening": "21:00 - 提醒记录今日数据",
    "review": "22:00 - 提醒生成复盘"
  }
}
```

## 技能能力

- `data.record` - 记录数据
- `data.query` - 查询数据
- `data.summary` - 生成汇总
- `data.review` - 生成复盘
- `data.trend` - 趋势分析
- `data.goal` - 目标管理
- `memory.write` - 沉淀长期记忆
