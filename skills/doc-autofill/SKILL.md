# 文档自动填写系统

根据模板自动填写文档，从长期记忆读取数据，告别重复劳动。

## 快速参考

| 场景 | 操作 |
|------|------|
| 填写表格 | `用我的信息填写这个报名表` |
| 批量生成 | `根据这个模板生成10份合同` |
| 自定义数据 | `用这些数据填写表格：姓名=张三，电话=13800138000` |
| 保存模板 | `把这个表格保存为模板` |

---

## 支持格式

| 格式 | 支持操作 |
|------|----------|
| Word (.docx) | 填写占位符、表格填充 |
| Excel (.xlsx) | 单格填充、批量数据 |
| Markdown (.md) | 模板替换 |
| PDF (.pdf) | 表单填写（需表单字段） |

---

## 占位符语法

### Word 模板

```
{{姓名}}
{{电话}}
{{日期}}
{{公司名称}}
```

### Excel 模板

| A | B |
|---|---|
| 姓名 | {{姓名}} |
| 电话 | {{电话}} |
| 日期 | {{日期}} |

## 数据源

### 1. 长期记忆

从 `reports/live_learning/user_preferences.json` 读取：

```json
{
  "personal_info": {
    "name": "张三",
    "phone": "13800138000",
    "email": "zhangsan@example.com",
    "company": "XX公司",
    "address": "北京市朝阳区..."
  }
}
```

### 2. 用户指定

```
用这些数据填写：
- 姓名：李四
- 电话：13900139000
- 公司：YY公司
```

### 3. 华为健康数据

```
用我的健康数据填写体检表
```

## 工作流

```
用户提供模板
    │
    ▼
┌─────────────────────────────────────┐
│  1. 解析模板                         │
│     - 识别占位符                     │
│     - 提取字段列表                   │
└─────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────┐
│  2. 匹配数据                         │
│     - 从长期记忆读取                 │
│     - 用户补充缺失字段               │
└─────────────────────────────────────┘
    │
    ▼
┌─────────────────────────────────────┐
│  3. 填写文档                         │
│     - 替换占位符                     │
│     - 保存新文件                     │
└─────────────────────────────────────┘
```

## 使用示例

### 填写报名表

```
用户: 用我的信息填写这个报名表
助手: 我检测到以下字段需要填写：
      - 姓名
      - 电话
      - 邮箱
      - 公司
      
      从记忆中找到：
      - 姓名：张三
      - 电话：13800138000
      - 邮箱：zhangsan@example.com
      - 公司：XX公司
      
      已生成填写完成的文档：报名表_已填写.docx
```

### 批量生成

```
用户: 根据这个合同模板，用这5个人的信息生成5份合同
助手: 请提供5个人的信息，格式如下：
      姓名,电话,身份证号
      张三,138xxx,110xxx
      李四,139xxx,120xxx
      ...
```

## 模板管理

```
用户: 把这个表格保存为"员工信息表"模板
```

```
用户: 我有哪些模板

      1. 员工信息表 - 4个字段
      2. 合同模板 - 8个字段
      3. 报销单 - 6个字段
```

## 与WPS对接

### 支持操作

| 操作 | 说明 |
|------|------|
| 打开文档 | 读取本地 .docx/.xlsx 文件 |
| 填写内容 | 替换占位符 |
| 保存文档 | 生成新文件 |
| 批量处理 | 一次生成多份 |

### 文件位置

- 模板目录: `reports/templates/`
- 输出目录: `reports/output/`

## 配置

### 个人信息模板

```json
{
  "personal_info": {
    "name": "",
    "phone": "",
    "email": "",
    "id_card": "",
    "company": "",
    "address": "",
    "bank_account": ""
  }
}
```

### 字段映射

```json
{
  "field_mapping": {
    "姓名": "name",
    "电话": "phone",
    "手机": "phone",
    "邮箱": "email",
    "身份证": "id_card",
    "公司": "company",
    "地址": "address"
  }
}
```

## 技能能力

- `doc.read` - 读取文档
- `doc.write` - 写入文档
- `doc.template` - 模板管理
- `memory.read` - 读取长期记忆
- `memory.write` - 保存数据到记忆
