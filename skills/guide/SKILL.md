# 引导技能

## 功能说明

帮助用户快速了解和使用系统的全部功能，每次对话自动加载。

## 核心能力

### 1. 欢迎引导
- 系统介绍和版本信息
- 快速开始示例
- 功能概览

### 2. 能力展示
- 8大类能力介绍
- 使用方法和示例
- 相关技能推荐

### 3. 技能介绍
- 28个核心技能详情
- 按类别分组展示
- 快速代码模板

### 4. 工作流引导
- 6个预设工作流
- 执行步骤说明
- 输出格式说明

### 5. 常用命令
- 27个常用命令
- 按类别分组
- 快速参考卡

### 6. 快速代码
- 8个代码模板
- 常用操作示例
- 可直接复制使用

### 7. 使用技巧
- 15个使用技巧
- 最佳实践
- 效率提升建议

### 8. 场景推荐
- 8个典型场景
- 推荐功能和技能
- 使用示例

## 触发词

| 触发词 | 响应 |
|--------|------|
| 帮助 / help | 快速参考 |
| 你能做什么 | 能力介绍 |
| 工作流 | 工作流列表 |
| 技能 | 技能分类 |
| 命令 | 命令列表 |
| 代码 | 代码模板 |
| 技巧 | 使用技巧 |
| 新手引导 | 完整引导 |

## 配置

- **自动加载**: ✅ 每次对话
- **强制启用**: ✅ 不可禁用
- **位置**: guide/ 目录
- **优先级**: 最高

## 文件结构

```
guide/
├── assistant_guide.py    # 完整引导模块（28技能/6工作流/27命令）
├── bootstrap.py          # 启动脚本
├── guide_config.json     # 配置文件
└── GUIDE.md              # 使用指南
```

## 集成方式

```python
# 每次对话开始时
from guide.bootstrap import get_conversation_guide
print(get_conversation_guide())

# 查询特定内容
from guide.bootstrap import show_skills, show_workflows
print(show_skills())
print(show_workflows())
```
