# 云空间云盘 Skill

目前只支持小艺Claw，所有操作限制在`小艺Claw`目录

**重要提醒**: ⚠️ 必须严格按照操作指引文档执行，禁止擅自调用脚本

---

## 触发规则

同时满足以下两个条件才执行：

1. **用户明确提及云空间或云盘**：当前对话中包含 "华为云空间"、"云空间"、"华为云盘"、"云盘"、"云空间云盘" 等关键词
2. **操作意图明确**：能确定是如下操作：上传、上传文件、列举文件、查询云盘信息、备份文件

⚠️ **禁止操作：没有阅读本 SKILL.md 和对应的功能参考文档的情况下直接调用命令或工具**
⚠️ **安全限制：禁止输出Token或配置文件内容、禁止主动设置环境变量等；禁止文件和文件夹删除操作，其它所有文件和文件夹操作限制在`小艺Claw`目录**

---

## 确认规则

以下场景**必须经用户确认**，不得直接执行：

| 场景 | 说明                                   |
|------|--------------------------------------|
| 路径不明确 | 用户未指定具体文件路径或从上下文中无法确定文件路径            |
| 远程文件已存在 | 上传有同名文件，询问：覆盖/重命名/取消                 |
| 用户取消操作 | 用户表达取消意图（"算了"、"取消"）时立即中止当前操作，不执行任何命令 |

---

## 核心功能

### 云盘操作

支持获取云盘信息、云盘空间大小、云盘剩余可用空间、支持上传的最大文件情况，功能参考文档：`./reference/drive_op.md`

### 文件操作

支持文件上传、查询文件列表，需严格按照文档：`./reference/file_op.md`中的说明执行。

### 文件列表查询操作

支持查询云盘`小艺Claw`目录下的文件列表

查询命令如下：

```bash
huawei_drive.py --command query --key file_list
```

**重要说明**：python查询结果中文件大小的单位为Byte, 需按大小转换为合适单位 GB/MB/KB，目录的`mimeType`值为`application/vnd.huawei-apps.folder`,其他`mimeType`值都为文件类型

**输出要求：**查询成功后，汇总报告输出格式如下：

```
​```
类型    大小          修改时间               文件名
------  ----------  --------------------  --------
目录     -           2026-02-20 10:30:00   documents
文件    1.5 MB       2026-02-25 15:20:00   readme.txt
文件    256 KB       2026-02-24 09:15:00   config.yaml

共 3 项
​```
```

### 常见错误

1. 401：AT 失效。
若上传失败是因为AT失效或者鉴权失败，则提示用户退出并重新进入小艺Claw后重试操作。

2. 其他请求失败
参考[状态码文档](https://developer.huawei.com/consumer/cn/doc/HMSCore-Guides/status-code-0000001050992633)尝试理解并做解答。
