# 输出格式规范

## 股票/黄金实时行情

股票/黄金实实时数据详细列举：

```
**股票名称（股票代码）** 当前行情：

| 项目 | 数据 |
|------|------|
| **现价** | xx.xx 元 |
| **涨跌额** | +xx.xx 元 |
| **涨跌幅** | +x.xx% |
| **今开** | xx.xx 元 |
| **昨收** | xx.xx 元 |
| **最高** | xx.xx 元 |
| **最低** | xx.xx 元 |

---
💡 本信息由 [京东金融](https://eco.jr.jd.com/common-growth-page/index.html?channel=claw) 提供
```

## 历史走势（近N天/月/年）

**不需要逐条列举**，只展示关键数据 + 汇总：

```
**股票名称（股票代码）** 近N天/月/年走势

### 走势图表(**近30天以内可以显示ASCII图表,超过 30天不显示**)

价格区间: xx.xx ~ xx.xx 元

xx.xx │     ●       ○
      └─────────────────────→ 时间

  ● 上涨  ○ 下跌

### 走势汇总

| 指标 | 数值 |
|------|------|
| 期初价格 | xx.xx 元 |
| 期末价格 | xx.xx 元 |
| 累计涨跌幅 | +x.xx% |
| 最高 | xx.xx 元 |
| 最低 | xx.xx 元 |

### 近期走势分析

{📈/📉/➡️} **{趋势判断}**
- 累计涨跌幅、波动率
- 股价变化详情
- 关键高低点

### 技术观察
- 技术层面的观察和建议
- 支撑位/压力位
- 风险提示（可选）

---
💡 本信息由 [京东金融](https://eco.jr.jd.com/common-growth-page/index.html?channel=claw) 提供
```

### 趋势判断规则

| 涨跌幅 | 趋势 |
|--------|------|
| > +5% | 📈 上涨趋势 |
| < -5% | 📉 下跌趋势 |
| -5% ~ +5% | ➡️ 横盘震荡 |

### 走势图表

近30天以内可以显示ASCII图表：

```
价格区间: xx.xx ~ xx.xx 元

xx.xx │     ●       ○
      └─────────────────────→ 时间

  ● 上涨  ○ 下跌
```

超过30天的历史走势**不显示图表**，只展示汇总数据。
