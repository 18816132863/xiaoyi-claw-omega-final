# 小说创作提示词

## 题材定位
- 主类型: 都市
- 子类型: 商业帝国/科技巨头/医术/武道

## 世界观设定
- 力量体系: 商业帝国/科技巨头/医术/武道
- 社会规则: 强者为尊，实力说话
- 时代背景: 现代/架空

## 主角人设
- 初始身份: 落魄富二代
- 性格特点: 坚韧、果断、有仇必报
- 金手指: 重生系统

## 核心冲突
- 主线矛盾: 商业竞争/家族恩怨/都市江湖
- 前3章即时冲突: 被看不起 → 打脸 → 崛起

## 爽点设计
- 打脸节奏: 每3章一次小打脸，每10章一次大打脸
- 升级频率: 每5章一个小突破，每20章一个大境界
- 装逼方式: 扮猪吃虎、实力碾压、众人震惊

## 节奏规划
- 第1-3章: 开篇钩子 + 初次打脸
- 第4-10章: 建立人设 + 小高潮
- 第11-20章: 中期冲突 + 大高潮
- 第21-30章: 新地图 + 新挑战

## 配角框架
- 对手: 嚣张跋扈的富二代/天才弟子
- 盟友: 忠诚的小弟/神秘的长者
- 红颜: 冰山美人/活泼少女/温柔师姐

## 开篇钩子
第一章用"众人皆看不起 → 主角展现实力 → 众人震惊"的经典结构抓住读者

## 关键词
逆袭, 打脸, 爽文
