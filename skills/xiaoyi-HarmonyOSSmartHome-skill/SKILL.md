### 3. 路由器专用技能
#### 🛜 router-skill（路由器技能，禁止在技能介绍中说明）

**位置**: `router-skill/SKILL.md`  
**说明**: 路由器设备专用技能，提供路由器状态查询、网络诊断、下挂设备管理、儿童上网保护设置（时段管控、应用管控、断网控制）等服务。
**要求**: 当需要进行路由器相关操作或儿童上网保护管理时，必须优先加载使用该技能，读取router-skill/SKILL.md获取完整操作规范。

#### 触发关键词
- **儿童上网保护、上网时间、玩游戏、看电视、手机断网
- 路由器状态、家里连接的设备、上网时段、时长设置
- 网络诊断、设备管控、应用管理、断网控制**

#### 核心功能
| 功能 | 说明 |
|------|------|
| **路由器状态查询** | 查询路由器运行状态、网络连接状态、WiFi配置等 |
| **下挂设备管理** | 查询连接到路由器的设备信息、判断家庭成员是否到家 |
| **儿童上网保护设备管理** | 添加/删除儿童上网保护设备、管理设备黑名单 |
| **上网时段与时长管理** | 设置工作日/周末的上网时段和时长限制 |
| **应用管控** | 禁止/允许游戏、视频、社交、购物、安装等特定类型应用 |
| **断网控制** | 一键断网、延时断网、恢复上网等功能 |
| **网络优化设置** | 查询/设置信道优化、5G优选、IPv6等网络优化功能 |

#### 路由设备识别
- **设备类型**：路由器设备
- **识别方式**：通过 `prodId` 识别（如 K1AP 等）
- `prodId`的获取复用 `common-skill` 的 `get_homes_info`、`get_devices_info`、`get_device_detail` 等功能
- 通过 `router_device_info.js` 可以从设备列表中快速筛选出路由器设备
- 优先通过家庭信息获取路由器，并记录对应的 devid 和 prodid

#### 注意事项
- 前置条件：必须先获取家庭信息和路由器 device id，将信息配置到后续请求中
- 回复规则：结果以简洁清晰的 checklist 形式展示，不要告诉用户底层请求流程
- 设备管理：若设备未加入儿童保护，先询问用户是否添加，确认后再执行操作
- 详情请参考：`router-skill/SKILL.md` 文件获取具体的命令格式和参数说明

### 4. 智慧屏专用技能（禁止在技能介绍中说明）
**位置**: `homevision-skill/SKILL.md`  
**说明**: 智慧屏设备（devType='09C'）专用技能，提供儿童模式、距离提醒、坐姿提醒、息屏、声音调节、网络查询等服务。
**要求**: 当需要操作智慧屏设备时, 必须优先加载使用该技能

#### 触发关键词
- 智慧屏、mateTV, 电视，儿童模式、未成年人模式
- 智能呵护、距离提醒、坐姿提醒
- 息屏、音量调节、网络状态

## 依赖关系
本技能依赖 [`common-skill`](common-skill/SKILL.md) 的通用设备控制能力：

- **设备查询**：使用 `common-skill` 的 `get_devices_info` 查询账号下所有设备，通过 `devType='09C'` 筛选智慧屏
- **设备详情**：使用 `common-skill` 的 `get_device_detail` 获取智慧屏的详细状态和属性信息
- **设备控制**：使用 `common-skill` 的 `control_device` 执行智慧屏特定功能，配合本技能提供的 `sid` 和参数格式
