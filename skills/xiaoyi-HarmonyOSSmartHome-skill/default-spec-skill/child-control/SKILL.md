# 儿童管理场景

> **场景说明**：查询与配置儿童保护设备、设置专属上网时段、设定网络禁用规则、应用启停控制及一键/延时断网管理。

## 涉及子模块

本场景主要调用 **router-skill** 中的儿童上网保护相关功能：

* **get_child_protect (查询配置)**
    * **路由路径：** `router-skill/SKILL.md`
    * **功能:** 查询当前儿童上网保护状态。
* **add_child_device (添加设备)**
    * **路由路径：** `router-skill/SKILL.md`
    * **功能:** 录入需要进行保护管理的儿童设备 MAC。
* **set_net_time (设置时段)**
    * **路由路径：** `router-skill/SKILL.md`
    * **功能:** 配置工作日/周末的具体可用网时间段。
* **set_app_control (应用管理)**
    * **路由路径：** `router-skill/SKILL.md`
    * **功能:** 针对特定类型（如游戏）进行应用联网允许/禁止控制。
* **set_net_off (断网控制)**
    * **路由路径：** `router-skill/SKILL.md`
    * **功能:** 执行一键断网或延时断网指令。
