# 门锁天气提醒设置功能Skill

## 1. 必读项

本技能用于**控制**门锁的天气提醒设置，开启后当所选位置预计有雨、雪、霾时，出门时会语音提示。**不支持查询操作**。

### 服务ID（sid）
- **sid**: `weatherReminder`

### 重要说明
- 当前仅对雨、雪、霾三种天气进行提示
- 需要设置合适的门锁留言、天气提醒音量，避免静音无法体验此场景
- 此场景需要门锁连接网络

## 2. 触发关键字

当用户提到以下关键词时，应加载本技能：
- 天气提醒
- 出门天气提醒
- 雨天提醒
- 下雪提醒

## 3. 用户语料示例

- "打开天气提醒"
- "关闭天气提醒"
- "设置天气提醒"
- "打开出门天气提醒"
- "关闭出门天气提醒"

## 4. 相关参数说明

### 4.1 天气提醒设置参数表

| 功能 | sid | 参数 | 参数说明 | 功能作用 |
|---|---|---|---|---|
| 天气提醒 | `weatherReminder` | `sw`, `st`, `et`, `lng`, `lat`, `city`, `ts` | sw：功能开关，0=关闭，1=打开<br/> st：天气提醒开始时间<br/>et：天气提醒结束事件<br/>leg：经度信息<br/>lat：维度信息<br/>city：城市行政编码<br/>ts：时间戳，配置时需要生成 | 1、所选位置预计有雨、雪、霾，出门时会语音提示，当前仅对雨、雪、霾三种天气进行提示<br/>2、需要设置合适的门锁留言、天气提醒音量，避免静音无法体验此场景<br/>3、此场景需要门锁连接网络 |

### 4.2 参数取值范围

| 参数 | 说明 |
|------|------|
| `sw` | 功能开关，0=关闭，1=打开 |
| `st` | 天气提醒开始时间，时间格式：HH:mm |
| `et` | 天气提醒结束时间，时间格式：HH:mm |
| `lng` | 经度信息 |
| `lat` | 纬度信息 |
| `city` | 城市行政编码 |
| `ts` | 时间戳，配置时需要生成，格式：20191226T075719Z |

## 5. 调用示例

### 5.1 打开天气提醒

```bash
node common-skill/bin/smarthome-claw.js control_device \
  --dev-id "xxx" \
  --prod-id "xxx" \
  --operation "POST" \
  --sid "weatherReminder" \
  --data '{
    "sw":	1,
	"st":	"06:00",
	"et":	"10:00",
	"lng":	"108.94",
	"lat":	"34.21",
	"city":	"610113",
	"ts":	"20260408T151137Z621"
}' \
  --verbose
```

### 5.2 关闭天气提醒

```bash

  --sid "weatherReminder" \
```

## 6. 注意事项

- 本技能仅支持控制，不支持查询操作
- 需要确保门锁连接网络才能正常使用天气提醒功能
- 需要设置合适的门锁音量避免静音无法体验
